## Тестирование создания категории объекта

### Описание

**Цель**: Проверка функциональности создания категории в модуле «Объекты», включая проверку негативных сценариев.

Тестирование создания категории с обязательными и необязательными полями, а также проверка обработки ошибок при неверном заполнении полей.

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Доступ администратора или суперадминистратора с разрешениями на просмотр объектов и создание объектов ("View Objects, Manage Objects").
- Доступ к изображениям для тестирования категорий.

### Тестовые данные
- Электронная почта: super@admin.com
- Пароль: [пароль из файла доступа в Notion]
- Наименование категории: "Тестовая категория"
- Описание категории: "Описание тестовой категории"
- Изображение: category_image.jpg
- Чек-бокс "Видимость для клиента": включен

### Шаги

#### Шаг 1:
Перейти на страницу создания категории.
**Ожидаемый результат**: Страница создания категории отображается корректно.

#### Шаг 2:
Заполнить обязательные поля: наименование категории, описание категории, изображение.
**Ожидаемый результат**: Поля принимают введенные данные.

#### Шаг 3:
Заполнить необязательное поле: чек-бокс "Видимость для клиента".
**Ожидаемый результат**: Поле принимает введенные данные.

#### Шаг 4:
Нажать кнопку "Создать".
**Ожидаемый результат**: Категория успешно создана и отображается в списке категорий.

### Негативные сценарии

#### Сценарий 1:
Не заполнять обязательные поля (наименование категории, описание категории, изображение).
**Ожидаемый результат**: Появляется сообщение об ошибке, указывающее на необходимость заполнения обязательных полей.

#### Сценарий 2:
Загрузить изображение неправильного формата (не изображение).
**Ожидаемый результат**: Появляется сообщение об ошибке, указывающее на неправильный формат изображения.

### Постусловия

- Удалить созданную тестовую категорию (если она была создана).
- Восстановить первоначальные настройки, если были изменены.

### Замечания/Примечания

- Убедиться, что изображение загружается корректно.
- Проверить, что чек-бокс "Видимость для клиента" правильно сохраняет свое состояние.
