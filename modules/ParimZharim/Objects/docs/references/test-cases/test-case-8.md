### Описание

**Цель**: Проверка функциональности таблицы "Теги объектов", включая сортировку, фильтрацию, выбор нескольких тегов и выполнение различных действий.

Тестирование отображения и работы элементов таблицы, таких как сортировка, фильтрация, выбор тегов и выполнение действий.

- Пользователь должен иметь доступ административной панели.

- Наличие созданных тегов объектов в системе для тестирования.
