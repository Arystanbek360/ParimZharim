## Тестирование создания тега объекта

### Описание

**Цель**: Проверка функциональности создания тега в модуле «Объекты», включая проверку негативных сценариев.

Тестирование создания тега с обязательными и необязательными полями, а также проверка обработки ошибок при неверном заполнении полей.

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Доступ администратора или суперадминистратора с разрешениями на просмотр объектов и создание объектов ("View Objects, Manage Objects").
- Доступ к изображению в формате SVG для тестирования тегов.

### Тестовые данные
- Электронная почта: super@admin.com
- Пароль: [пароль из файла доступа в Notion]
- Наименование тега: "Тестовый тег"
- Изображение: test_tag.svg
- Чек-бокс "Видимость для клиента": включен

### Шаги

#### Шаг 1:
Перейти на страницу создания тега.
**Ожидаемый результат**: Страница создания тега отображается корректно.

#### Шаг 2:
Заполнить обязательные поля: наименование тега, изображение (формат SVG).
**Ожидаемый результат**: Поля принимают введенные данные.

#### Шаг 3:
Заполнить необязательное поле: чек-бокс "Видимость для клиента".
**Ожидаемый результат**: Поле принимает введенные данные.

#### Шаг 4:
Нажать кнопку "Создать".
**Ожидаемый результат**: Тег успешно создан и отображается в списке тегов.

### Негативные сценарии

#### Сценарий 1:
Не заполнять обязательные поля (наименование тега, изображение).
**Ожидаемый результат**: Появляется сообщение об ошибке, указывающее на необходимость заполнения обязательных полей.

#### Сценарий 2:
Загрузить изображение неправильного формата (не SVG).
**Ожидаемый результат**: Появляется сообщение об ошибке, указывающее на неправильный формат изображения.

### Постусловия

- Удалить созданный тестовый тег (если он был создан).
- Восстановить первоначальные настройки, если были изменены.

### Замечания/Примечания

- Убедиться, что изображение загружается корректно и соответствует формату SVG.
- Проверить, что чек-бокс "Видимость для клиента" правильно сохраняет свое состояние.
