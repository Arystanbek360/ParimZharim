
## Получение активного заказа через API

### Описание

**Цель**: Тестирование получения активного заказа через API для авторизованного клиента.

Проверка процесса получения активного заказа клиента и обработка ошибок для различных сценариев.

### Предусловия и необходимые инструменты

- Авторизованный клиент с профилем клиента, имеющий хотя бы один активный заказ (не в статусе «отменен», «время истекло», «выполнен»).
- Доступ к API по адресу /api/orders/get-active-order.
- Токен авторизованного клиента.

### Тестовые данные
- Токен клиента с активными заказами: [токен клиента с активными заказами]
- Токен клиента без активных заказов: [токен клиента без активных заказов]
- Неверный или отсутствующий токен: [неверный токен]

### Позитивные кейсы

#### Шаг 1:
Отправить GET-запрос на /api/orders/get-active-order с токеном клиента с активными заказами.
**Ожидаемый результат**: Возвращается статус-код 200 и ID первого активного заказа:
```json
{
  "active_order": 123
}
```

#### Шаг 2:
Отправить GET-запрос на /api/orders/get-active-order с токеном клиента без активных заказов.
**Ожидаемый результат**: Возвращается статус-код 200 и значение null:
```json
{
  "active_order": null
}
```

### Негативные кейсы

#### Шаг 1:
Отправить GET-запрос на /api/orders/get-active-order без токена или с неверным токеном.
**Ожидаемый результат**: Возвращается статус-код 401 с сообщением об ошибке "Unauthorized".

### Примеры запросов и ответов для негативных кейсов

#### Без токена или с неверным токеном:
GET /api/orders/get-active-order
Headers:
```
Authorization: Bearer [неверный токен]
```
**Ожидаемый результат**:
```json
{
 "error": "Unauthorized"
}
```

### Постусловия

- Убедиться, что клиент может повторно авторизоваться при необходимости.
- Убедиться, что данные активного заказа корректно отображаются для клиентов.

### Замечания/Примечания

- Проверить корректность работы авторизации через токен.
- Убедиться, что система корректно обрабатывает запросы для клиентов без активных заказов.
- Проверить, что неавторизованные пользователи получают ошибку 401.
