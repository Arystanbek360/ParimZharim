# Принципы интеграции IIKO в ПЖ

1. Есть сервис IikoIntegrationService, который предоставляет методы для работы с IIKO.
2. Сервис должен уметь: а) выгружать заказ в IIKO б) синхронизировать список блюд из IIKO в отдельную таблицу
3. При редактировании блюда можно указать связь с товаром из IIKO
