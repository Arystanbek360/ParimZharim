
## Проверка API запроса категорий меню

### Описание

**Цель**: Тестирование API для получения списка категорий меню

Проверка работы API запроса, который возвращает список категорий меню, чтобы убедиться, что запрос выполняется корректно и возвращает ожидаемый результат.

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ к API и валидный токен авторизации (если требуется).
- Установленный инструмент для тестирования API (например, Postman или curl).

### Тестовые данные

- URL запроса: `{{host}}/api/products-services/get-product-categories`

### Шаги

#### Шаг 1:
Отправить GET-запрос на URL `{{host}}/api/products-services/get-product-categories`.
**Ожидаемый результат**: Статус ответа 200 ОК.

#### Шаг 2:
Проверить формат и содержимое ответа.
**Ожидаемый результат**: Ответ в формате JSON со списком категорий продуктов и услуг:
```json
[
    {
        "id": 6,
        "name": "Фаст-фуд"
    },
    {
        "id": 1,
        "name": "Хит продаж"
    },
    {
        "id": 3,
        "name": "Супы"
    },
    {
        "id": 4,
        "name": "Мясо"
    },
    {
        "id": 2,
        "name": "Салаты"
    }
]
```

### Постусловия

- Нет

### Замечания/Примечания

- Убедитесь, что сервер запущен и доступен для отправки запросов.
- Проверьте, что у вас есть необходимые права доступа для выполнения API запросов.
