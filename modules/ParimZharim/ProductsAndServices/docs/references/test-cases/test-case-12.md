## Проверка создания категории меню

### Описание

**Цель**: Тестирование функции создания новой категории меню.

Проверка работы формы создания категории меню, включая все поля для ввода данных, чек-бокс, функциональность блока "сервисы", а также функциональность кнопок "Отмена", "Создать и добавить другой" и "Создать категория меню".

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Пользователь должен быть зарегистрирован в системе и иметь доступ к странице создания категории меню.
- Установленный веб-браузер с доступом к веб-приложению.

### Тестовые данные

- Электронная почта: super@admin.com
- Пароль: [пароль из файла доступа в Notion]
- Название категории: "Десерты"
- Видимость для клиента: Да

### Шаги

#### Шаг 1:
Перейти на страницу создания категории меню.
**Ожидаемый результат**: Страница создания категории меню отображается корректно, видны все поля для ввода данных и чек-бокс.

#### Шаг 2:
Заполнить поле "Название категории" значением "Десерты".
**Ожидаемый результат**: Поле "Название категории" принимает введенное значение.

#### Шаг 3:
Установить чек-бокс "Видимость для клиента".
**Ожидаемый результат**: Чек-бокс отмечен.

#### Шаг 4:
Нажать на кнопку "Отмена".
**Ожидаемый результат**: Создание категории меню отменено, пользователь возвращается на страницу таблицы категорий меню.

#### Шаг 5:
Заполнить все поля заново и нажать кнопку "Создать и добавить другой".
**Ожидаемый результат**: Категория меню сохранена, страница обновляется для создания новой категории меню.

#### Шаг 6:
Заполнить все поля заново и нажать кнопку "Создать категория меню".
**Ожидаемый результат**: Категория меню сохранена, пользователь перенаправлен на страницу просмотра созданного элемента категории.

#### Шаг 7:
Нажать на сворачиваемый блок "сервисы".
**Ожидаемый результат**: Блок разворачивается, отображаются таблица привязанных услуг и соответствующие элементы управления.

#### Шаг 8:
В таблице услуг выбрать несколько элементов с помощью чек-боксов.
**Ожидаемый результат**: Услуги выделяются.

#### Шаг 9:
Нажать на три точки рядом с элементом услуги и выбрать "Создать копию".
**Ожидаемый результат**: Открывается страница создания услуги с данными, скопированными из выбранного элемента.

#### Шаг 10:
Нажать на три точки рядом с элементом услуги и выбрать "Просмотр".
**Ожидаемый результат**: Открывается страница просмотра элемента без возможности редактирования.

#### Шаг 11:
Нажать на три точки рядом с элементом услуги и выбрать "Редактировать".
**Ожидаемый результат**: Открывается страница редактирования выбранного элемента.

#### Шаг 12:
Нажать на три точки рядом с элементом услуги и выбрать "Удалить".
**Ожидаемый результат**: Появляется окно подтверждения удаления.

### Постусловия

- Удалить созданные тестовые данные, если они были добавлены в ходе тестирования.
- Восстановить исходные настройки и данные в таблице категорий меню.
