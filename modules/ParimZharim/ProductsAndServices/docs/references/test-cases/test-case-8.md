## Проверка создания услуги

### Описание

**Цель**: Тестирование функции создания новой услуги.

Проверка работы формы создания услуги, включая все поля для ввода данных, чек-бокс, загрузку изображения, выбор и добавление категории, а также функциональность кнопок "Отмена", "Создать и добавить другой" и "Создать меню".

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Пользователь должен быть зарегистрирован в системе и иметь доступ к странице создания услуги.
- Установленный веб-браузер с доступом к веб-приложению.

- Электронная почта: super@admin.com

- Цена: 1500
- Включен: Да
- Изображение: "massage.jpg"
- Категория: "Оздоровительные услуги"

### Шаги

#### Шаг 1:
Перейти на страницу создания услуги.
**Ожидаемый результат**: Страница создания услуги отображается корректно, видны все поля для ввода данных, чек-бокс, поле для загрузки изображения и выпадающий список категорий.

#### Шаг 2:
Заполнить поле "Название" значением "Массаж".
**Ожидаемый результат**: Поле "Название" принимает введенное значение.

#### Шаг 3:
Заполнить поле "Описание" значением "Расслабляющий массаж всего тела".
**Ожидаемый результат**: Поле "Описание" принимает введенное значение.

#### Шаг 4:
Заполнить поле "Цена" значением "1500".
**Ожидаемый результат**: Поле "Цена" принимает введенное значение.

#### Шаг 5:
Установить чек-бокс "Включен".
**Ожидаемый результат**: Чек-бокс отмечен.

#### Шаг 6:
Загрузить изображение, выбрав файл "massage.jpg".
**Ожидаемый результат**: Изображение загружено и отображается в поле.

#### Шаг 7:
Выбрать категорию "Оздоровительные услуги" из выпадающего списка.
**Ожидаемый результат**: Категория "Оздоровительные услуги" выбрана.

#### Шаг 8:
Нажать на кнопку "плюс" справа от выпадающего списка категорий и добавить новую категорию "Косметические услуги".
**Ожидаемый результат**: Новая категория "Косметические услуги" добавлена в выпадающий список категорий.

#### Шаг 9:
Нажать на кнопку "Отмена".
**Ожидаемый результат**: Создание услуги отменено, пользователь возвращается на страницу таблицы услуг.

#### Шаг 10:
Заполнить все поля заново и нажать кнопку "Создать и добавить другой".
**Ожидаемый результат**: Услуга сохранена, страница обновляется для создания новой услуги.

#### Шаг 11:
Заполнить все поля заново и нажать кнопку "Создать меню".
**Ожидаемый результат**: Услуга сохранена, пользователь перенаправлен на страницу просмотра созданного элемента услуги.

### Постусловия

- Удалить созданные тестовые данные, если они были добавлены в ходе тестирования.
- Восстановить исходные настройки и данные в таблице услуг.
