## Проверка создания меню

### Описание

**Цель**: Тестирование функции создания нового элемента меню.

Проверка работы формы создания меню, включая все поля для ввода данных, чек-бокс, загрузку изображения, выбор и добавление категории, а также функциональность кнопок "Отмена", "Создать и добавить другой" и "Создать меню".

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Доступ администратора или суперадминистратора с разрешениями на просмотр и редактирование таблицы меню.
- Установленный веб-браузер с доступом к веб-приложению.

### Тестовые данные

- Электронная почта: super@admin.com
- Пароль: [пароль из файла доступа в Notion]
- Название: "Омлет"
- Описание: "Классический омлет с зеленью"
- Цена: 200
- Включен: Да
- Изображение: "omelet.jpg"
- Категория: "Завтрак"

### Шаги

#### Шаг 1:
Перейти на страницу создания меню.
**Ожидаемый результат**: Страница создания меню отображается корректно, видны все поля для ввода данных, чек-бокс, поле для загрузки изображения и выпадающий список категорий.

#### Шаг 2:
Заполнить поле "Название" значением "Омлет".
**Ожидаемый результат**: Поле "Название" принимает введенное значение.

#### Шаг 3:
Заполнить поле "Описание" значением "Классический омлет с зеленью".
**Ожидаемый результат**: Поле "Описание" принимает введенное значение.

#### Шаг 4:
Заполнить поле "Цена" значением "200".
**Ожидаемый результат**: Поле "Цена" принимает введенное значение.

#### Шаг 5:
Установить чек-бокс "Включен".
**Ожидаемый результат**: Чек-бокс отмечен.

#### Шаг 6:
Загрузить изображение, выбрав файл "omelet.jpg".
**Ожидаемый результат**: Изображение загружено и отображается в поле.

#### Шаг 7:
Выбрать категорию "Завтрак" из выпадающего списка.
**Ожидаемый результат**: Категория "Завтрак" выбрана.

#### Шаг 8:
Нажать на кнопку "плюс" справа от выпадающего списка категорий и добавить новую категорию "Обеды".
**Ожидаемый результат**: Новая категория "Обеды" добавлена в выпадающий список категорий.

#### Шаг 9:
Нажать на кнопку "Отмена".
**Ожидаемый результат**: Создание меню отменено, пользователь возвращается на страницу таблицы меню.

#### Шаг 10:
Заполнить все поля заново и нажать кнопку "Создать и добавить другой".
**Ожидаемый результат**: Меню сохранено, страница обновляется для создания нового элемента меню.

#### Шаг 11:
Заполнить все поля заново и нажать кнопку "Создать меню".
**Ожидаемый результат**: Меню сохранено, пользователь перенаправлен на страницу просмотра созданного элемента меню.

### Постусловия

- Удалить созданные тестовые данные, если они были добавлены в ходе тестирования.
- Восстановить исходные настройки и данные в таблице меню.
