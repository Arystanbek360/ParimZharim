## Проверка создания категории услуги

### Описание

**Цель**: Тестирование функции создания новой категории услуги.

Проверка работы формы создания категории услуги, включая все поля для ввода данных, чек-бокс, а также функциональность кнопок "Отмена", "Создать и добавить другой" и "Создать категория меню".

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ административной панели.
- Пользователь должен быть зарегистрирован в системе и иметь доступ к странице создания категории услуги.
- Установленный веб-браузер с доступом к веб-приложению.

### Тестовые данные

- Электронная почта: super@admin.com
- Пароль: [пароль из файла доступа в Notion]
- Название категории: "Спа услуги"
- Видимость для клиента: Да

### Шаги

#### Шаг 1:
Перейти на страницу создания категории услуги.
**Ожидаемый результат**: Страница создания категории услуги отображается корректно, видны все поля для ввода данных и чек-бокс.

#### Шаг 2:
Заполнить поле "Название категории" значением "Спа услуги".
**Ожидаемый результат**: Поле "Название категории" принимает введенное значение.

#### Шаг 3:
Установить чек-бокс "Видимость для клиента".
**Ожидаемый результат**: Чек-бокс отмечен.

#### Шаг 4:
Нажать на кнопку "Отмена".
**Ожидаемый результат**: Создание категории услуги отменено, пользователь возвращается на страницу таблицы категорий услуг.

#### Шаг 5:
Заполнить все поля заново и нажать кнопку "Создать и добавить другой".
**Ожидаемый результат**: Категория услуги сохранена, страница обновляется для создания новой категории услуги.

#### Шаг 6:
Заполнить все поля заново и нажать кнопку "Создать категория меню".
**Ожидаемый результат**: Категория услуги сохранена, пользователь перенаправлен на страницу просмотра созданного элемента категории.

### Постусловия

- Удалить созданные тестовые данные, если они были добавлены в ходе тестирования.
- Восстановить исходные настройки и данные в таблице категорий услуг.
