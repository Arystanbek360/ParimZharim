
## Тестирование API для получения полного списка меню с указанием категорий

### Описание

**Цель**: Проверить возможность получения полного списка меню с указанием категорий через API.

Проверка корректности работы API по запросу полного списка меню и проверки структуры ответа.

### Предусловия и необходимые инструменты

- Доступ к API с установленными правами для запроса полного списка меню
- Инструмент для тестирования API, например, Postman

### Тестовые данные
- URL для запроса: {{host}}/api/products-services/get-products-by-category

### Шаги

#### Шаг 1:
Отправить GET запрос на URL: {{host}}/api/products-services/get-products-by-category.
**Ожидаемый результат**: Статус ответа 200 ОК и тело ответа в формате JSON.
