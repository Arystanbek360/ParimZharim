
## Тестирование API для получения списка услуг по категориям

### Описание

**Цель**: Проверить возможность получения списка услуг по категориям через API.

Проверка корректности работы API по запросу списка услуг и проверки структуры ответа.

### Предусловия и необходимые инструменты

- Доступ к API с установленными правами для запроса списка услуг по категориям
- Инструмент для тестирования API, например, Postman

### Тестовые данные
- URL для запроса: {{host}}/api/products-services/get-services-by-category

### Шаги

#### Шаг 1:
Отправить GET запрос на URL: {{host}}/api/products-services/get-services-by-category.
**Ожидаемый результат**: Статус ответа 200 ОК и тело ответа в формате JSON.
