## Смена номера телефона в мобильном приложении

### Описание

**Цель**: Тестирование функции смены номера телефона в мобильном приложении, включая авторизацию, ввод нового номера телефона, получение и ввод кода подтверждения, и проверку отображения обновленных данных.

Тестирование включает проверку смены номера телефона зарегистрированного пользователя и отображение обновленных данных в разделе «Личная информация».

### Предусловия и необходимые инструменты

- Установленное мобильное приложение.
- Профиль пользователя, зарегистрированный в системе (может быть создан в test-case-8, test-case-9).
- Реальный номер телефона для получения СМС.

### Тестовые данные
- Старый номер телефона пользователя: [Старый номер телефона]
- Новый тестовый номер телефона: +7 0000000777 с кодом 666666
- Новый реальный номер телефона: [Реальный номер России или Казахстана для получения СМС]

### Шаги

#### Шаг 1:
Открыть мобильное приложение и авторизоваться с использованием учетных данных.
**Ожидаемый результат**: Пользователь успешно авторизуется и попадает на главную страницу.

#### Шаг 2:
Перейти в раздел «Профиль», затем в «Личная информация».
**Ожидаемый результат**: Пользователь видит текущие данные профиля, включая старый номер телефона.

#### Шаг 3:
Ввести новый номер телефона в поле «Телефон» и нажать на кнопку «Получить код».
**Ожидаемый результат**: Пользователь получает СМС с кодом подтверждения.

#### Шаг 4:
Ввести код подтверждения (666666 для тестового номера или код из СМС для реального номера) и нажать кнопку «Изменить номер».
**Ожидаемый результат**: Появляется сообщение о успешном изменении номера телефона, и пользователь возвращается на экран профиля.

#### Шаг 5:
Проверить отображение обновленного номера телефона на экране профиля.
**Ожидаемый результат**: Обновленный номер телефона отображается корректно.

#### Шаг 6:
Перейти в раздел «Личная информация» и проверить отображение обновленного номера телефона.
**Ожидаемый результат**: Обновленный номер телефона отображается корректно в разделе «Личная информация».

### Постусловия

- Очистить тестовые данные из системы при необходимости.
- Восстановить настройки приложения при необходимости.

### Замечания/Примечания

- Тестирование предполагает наличие корректного номера телефона и возможности получения СМС.
- Для проверки отображения данных может потребоваться доступ к реальному мобильному устройству.
