## Тестирование регистрации в мобильном приложении

### Описание

**Цель**: Тестирование процесса регистрации нового пользователя в мобильном приложении, включая проверку обязательных полей, отображения данных, и редактирования имени.

Тестирование включает проверку регистрации нового пользователя с использованием тестового или реального номера телефона, заполнение обязательных полей, отображение и редактирование данных пользователя.

### Предусловия и необходимые инструменты

- Установленное мобильное приложение.
- Новый или тестовый номер телефона пользователя, который еще не регистрировался в системе.
- Тестовый номер: +7000000XXXX с кодом 666666.
- Реальный номер России или Казахстана для получения СМС.

### Тестовые данные
- Имя: "Иван Иванов"
- Дата рождения: "2000-01-01" (формат YYYY-MM-DD)
- Новый номер телефона: +7000000XXXX

### Шаги

#### Шаг 1:
Открыть мобильное приложение и ввести новый номер телефона на экране регистрации.
**Ожидаемый результат**: Пользователь получает СМС с кодом подтверждения.

#### Шаг 2:
Ввести код подтверждения на экране подтверждения.
**Ожидаемый результат**: Пользователь переходит на экран ввода данных.

#### Шаг 3:
На экране ввода данных заполнить поле ФИО.
**Ожидаемый результат**: Поле ФИО принимает введенные данные.

#### Шаг 4:
Заполнить поле даты рождения.
**Ожидаемый результат**: Поле даты рождения принимает введенные данные.

#### Шаг 5:
Попробовать перейти к следующему экрану без заполнения одного из обязательных полей.
**Ожидаемый результат**: Пользователь не может перейти к следующему экрану, пока все обязательные поля не заполнены.

#### Шаг 6:
Заполнить поле даты рождения датой, которая меньше 18 лет (например, "2010-01-01").
**Ожидаемый результат**: Появляется сообщение об ошибке «Вам должно быть 18 лет для регистрации».

#### Шаг 7:
После заполнения всех обязательных полей нажать кнопку продолжения.
**Ожидаемый результат**: Появляется сообщение «Аккаунт зарегистрирован», пользователь оказывается на главной странице.

#### Шаг 8:
Перейти в раздел «Личная информация» и проверить отображение данных.
**Ожидаемый результат**: Данные пользователя отображаются корректно.

#### Шаг 9:
Нажать на карандашик для редактирования имени, изменить имя и нажать «Сохранить изменения».
**Ожидаемый результат**: Измененное имя отображается корректно в профиле.

#### Шаг 10:
Попробовать редактировать дату рождения.
**Ожидаемый результат**: Поле даты рождения недоступно для редактирования.

### Постусловия

- Очистить тестовые данные из системы при необходимости.
- Восстановить настройки приложения при необходимости.

### Замечания/Примечания

- Тестирование предполагает наличие корректного номера телефона и возможности получения СМС.
- Для проверки отображения данных может потребоваться доступ к реальному мобильному устройству.
