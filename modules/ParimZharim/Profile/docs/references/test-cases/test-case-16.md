## Тестирование запроса на удаление учетной записи

### Описание

**Цель**: Тестирование процесса запроса на удаление учетной записи и получения уведомления в административной панели.

Проверка корректности работы функции запроса на удаление учетной записи через веб-интерфейс, а также получение соответствующего уведомления в административной панели.

### Предусловия и необходимые инструменты

- Зарегистрированная учетная запись в системе.
- Доступ к веб-интерфейсу для отправки запроса на удаление учетной записи.
- Администраторская учетная запись с доступом к административной панели.

### Тестовые данные
- Зарегистрированная учетная запись (взять для примера из административной панели)
- Ссылка для зявки удаления пользователя: https://develop.parimzharim.devcraft.meteoro.ai/delete-account

### Шаги

#### Шаг 1:
Перейти по предоставленной ссылке для запроса на удаление учетной записи.
**Ожидаемый результат**: Открывается страница с формой для ввода номера учетной записи.

#### Шаг 2:
Ввести номер телефона зарегистрированной учетной записи  в соответствующее поле.
**Ожидаемый результат**: Поле для ввода данных принимает корректный номер учетной записи.

#### Шаг 3:
Отправить запрос на удаление учетной записи.
**Ожидаемый результат**: Запрос успешно отправлен, отображается подтверждающее сообщение.

#### Шаг 4:
Перейти в административную панель и проверить уведомления в правой верхней части.
**Ожидаемый результат**: В административной панели отображается уведомление с текстом.
```
Запрос на удаление аккаунта. Номер телефона: *указанный в заявке номер*
```
### Постусловия

- Убедиться, что запрос на удаление учетной записи был обработан вручную администратором.
- В случае необходимости восстановить учетную запись для дальнейшего тестирования.

### Замечания/Примечания

- Убедитесь, что уведомление в административной панели отображается корректно и содержит указанный текст.
- Проверьте, что уведомление видно только администраторам с соответствующими правами доступа.
