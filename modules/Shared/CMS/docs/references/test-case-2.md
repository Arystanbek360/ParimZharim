## Тестирование изменения контента конфиденциальности

### Описание

**Цель**: Тестирование функции редактирования контента на странице конфиденциальности через модуль CMS.

Тестирование процесса изменения контента на странице конфиденциальности, включающее в себя переход на страницу редактирования контента, внесение изменений и проверку обновлений на странице по ссылке.

### Предусловия и необходимые инструменты

- Доступ к модулю CMS в системе.
- Наличие учетной записи с правами на редактирование контента.

### Тестовые данные
- Страница для редактирования: {{host}}/privacy-policy
- Текущий контент страницы конфиденциальности.

### Шаги

#### Шаг 1:
Перейти на страницу модуля CMS в раздел "Контент".
**Ожидаемый результат**: Страница раздела "Контент" загружена корректно.

#### Шаг 2:
Найти и нажать кнопку "Редактировать" рядом с контентом страницы конфиденциальности.
**Ожидаемый результат**: Открывается страница редактирования контента.

#### Шаг 3:
Внести изменения в текст контента.
**Ожидаемый результат**: Поля для ввода данных позволяют редактировать текст.

#### Шаг 4:
Нажать кнопку "Обновить" для сохранения изменений.
**Ожидаемый результат**: Изменения успешно сохранены, отображается сообщение о успешном обновлении.

#### Шаг 5:
Перейти на страницу по ссылке {{host}}/privacy-policy.
**Ожидаемый результат**: Страница загружается с отображением внесенных изменений.

#### Шаг 6:
Перейти в мобильном приложении на страницу профиль, открыть раздел "Договор оферты"
**Ожидаемый результат**: Страница загружается с отображением внесенных изменений.

### Постусловия

- Проверить, что изменения на странице конфиденциальности соответствуют внесенным в модуле CMS.
- В случае необходимости восстановить первоначальный контент страницы конфиденциальности.

### Замечания/Примечания

- Убедитесь, что изменения контента видны только после обновления страницы.
- Проверьте, что обновленный контент отображается корректно на всех устройствах и браузерах.
