## Создание роли

### Описание

**Цель**: Проверка функциональности создания новой роли, привязки разрешений к этой роли, а также проверка обработки ошибок при создании роли с некорректными данными.

Пользователь должен иметь возможность создать новую роль, заполнив обязательное поле "Название", и увидеть страницу с деталями роли, включая автоматически присвоенный ID, введенное название и возможность прикрепления разрешений. Также должны быть проверены негативные сценарии, связанные с некорректным заполнением формы.

### Предусловия и необходимые инструменты

- Пользователь должен иметь доступ к админ-панели.
- Пользователь должен обладать правами на добавление ролей.
- Необходимо наличие активной сессии пользователя.

### Тестовые данные
- Название роли: TestRole
- Некорректное название роли: (пустое поле)

### Шаги

#### Шаг 1:
Зайти в админ-панель.
**Ожидаемый результат**: Пользователь успешно авторизуется в админ-панели.

#### Шаг 2:
Перейти в раздел "Роли".
**Ожидаемый результат**: Открывается страница с таблицей созданных ролей.

#### Шаг 3:
Нажать кнопку "Создать роль".
**Ожидаемый результат**: Открывается страница создания новой роли с полем для ввода "Название".

#### Шаг 4:
Заполнить поле "Название" значением "TestRole" и нажать кнопку "Создать Роль".
**Ожидаемый результат**: Открывается страница просмотра роли, где отображается блок с деталями роли, включая ID роли и введенное название.

#### Шаг 5:
Проверить наличие блока разрешений на странице просмотра роли.
**Ожидаемый результат**: Блок разрешений присутствует, и есть возможность прикрепить разрешения к роли.

#### Негативный сценарий 1:
На странице создания роли оставить поле "Название" пустым и нажать кнопку "Создать Роль".
**Ожидаемый результат**: Появляется сообщение об ошибке, информирующее пользователя о необходимости заполнить обязательное поле "Название". Роль не создается.

#### Негативный сценарий 2:
Пользователь без прав на доступ к разделу "Роли" и на добавление ролей пытается создать роль.
**Ожидаемый результат**: Раздел "Роли" и соответствующие элементы управления, такие как кнопка "Создать роль", не отображаются для пользователя.

### Постусловия

- Удалить тестовую роль "TestRole", чтобы очистить тестовые данные.
