# Документация по переменным окружения для модуля оплаты CloudPaymentsService

## Введение

В этом разделе описаны переменные окружения, используемые для интеграции с платежной системой CloudPayments в рамках модуля оплаты. Переменные окружения используются для конфигурации ключевых параметров, необходимых для работы сервиса.

## Переменные окружения

### `PAYMENT_CLOUDPAYMENTS_PUBLIC_ID`

- **Описание**: Публичный идентификатор, выданный CloudPayments, используется для идентификации вашего аккаунта в системе CloudPayments.
- **Пример значения**: `pk_1234567890abcdef1234567890abcdef`
- **Использование в коде**:
  ```php
  'payment_cloudpayments_public_id' => env('PAYMENT_CLOUDPAYMENTS_PUBLIC_ID', ''),
  ```
  Данная переменная извлекается из файла окружения и используется для конфигурации модуля оплаты в файле `config/app.php` или аналогичном.

### `PAYMENT_CLOUDPAYMENTS_SECRET`

- **Описание**: Секретный ключ, используемый для авторизации запросов к API CloudPayments и защиты транзакций от неавторизованного доступа.
- **Пример значения**: `sk_1234567890abcdef1234567890abcdef`
- **Использование в коде**:
  ```php
  'payment_cloudpayments_secret' => env('PAYMENT_CLOUDPAYMENTS_SECRET', ''),
  ```
  Эта переменная загружается из `.env` файла и применяется для настройки соединения с платежным сервисом.

## Примеры использования в приложении

Извлечение и использование данных из конфигурационных файлов осуществляется следующим образом:

```php
$this->publicId = config('app.payment_cloudpayments_public_id');
$this->apiSecret = config('app.payment_cloudpayments_secret');
```

В данном примере значения `publicId` и `apiSecret` извлекаются из конфигурационного файла и используются в методах класса, управляющего платежами, для выполнения транзакций и других операций, связанных с платежной системой.

## Заключение

Настройка и управление ключевыми параметрами через переменные окружения обеспечивает безопасность и удобство конфигурации платежного модуля. Рекомендуется хранить эти ключи в защищенном хранилище и исключать `.env` файл из систем контроля версий для предотвращения утечек конфиденциальной информации.
