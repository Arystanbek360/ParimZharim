### Разрешения (Permissions)

Разрешения - это способ управления доступом к функционалу приложения. В зависимости от роли пользователя, ему могут быть доступны различные функции приложения. Например, администратору доступны все функции приложения, а пользователю только часть из них.

#### Список разрешений

- `VIEW_NOTIFICATIONS` - просмотр уведомлений
- `MANAGE_NOTIFICATIONS` - управление уведомлениями
